## About CareVault

CareVault is a comprehensive medical information management system designed specifically for chronic disease patients and their caregivers. Built with Next.js and Supabase, this full-stack application enables efficient management of medical information, medication records, appointment reminders, and important documents.

### Key Features
- Multi-user management (Caregiver and Patient roles)
- Care recipient management with comprehensive profiles
- Medication tracking and management
- Appointment scheduling and reminders
- Document storage and organization
- Emergency summary generation
- Role-based access control with data isolation

### Technology Stack
- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **Deployment**: Vercel
- **UI Components**: Custom components following shadcn/ui patterns
- **Icons**: Lucide React
