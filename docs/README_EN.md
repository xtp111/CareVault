# Caregiver Secure Documentation App

This project is a full‑stack application designed for caregivers to securely store, manage, and share critical information for people with chronic conditions.

## Core Features
- Secure document storage (legal, medical, financial, identification)
- Structured medical records (doctors, medications, conditions)
- Emergency Summary Generator (one‑page, read‑only, PDF/shareable link)
- Role‑based access (caregiver, emergency viewer)
- Audit logging

This repository contains backend, frontend, and documentation suitable for an academic group project.
